# Screenshot Guide

Use this folder for polished screenshots before publishing the repository.

Recommended screenshots:

- `assets/screenshots/live-coach.png`
- `assets/screenshots/photo-analysis.png`
- `assets/screenshots/progress-dashboard.png`
- `assets/screenshots/pose-library.png`

Suggested capture checklist:

- Use the same iPhone simulator/device frame for all screenshots.
- Avoid showing private notes, local file paths, or unfinished debug UI.
- Use clear pose examples with good lighting.
- Keep the screenshots cropped consistently.
- Update the README table after adding the final images.
